About This Platform

Built with Purpose, Powered by MERN

A real-world, production-grade learning platform engineered from scratch.

I developed a Full Stack Online Learning Platform using the MERN Technology Stack — MongoDB, Express.js, React.js, and Node.js. The platform is designed to deliver structured educational content with secure authentication, real-time tracking, and a seamless experience across all devices.

Additionally, I carried out data analysis using Python, SQL, and Excel to surface actionable insights on user engagement, course performance, and platform growth — enabling better, data-driven decisions.

The system follows MVC architecture, implements JWT-based authentication, and uses a RESTful API with Mongoose ODM for clean separation of concerns across all layers.
